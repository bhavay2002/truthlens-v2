# 🧠 Step 1 — Define metric

You have multiple metrics:

```python
val_metrics = {
    'emotion_score': ...,
    'ideology_score': ...,
    ...
}
```

👉 Create a **single scalar**:

```python
def compute_overall_score(metrics: dict):
    return sum(metrics.values()) / len(metrics)
```

---

# 🔥 Step 2 — Modify Trainer Loop

## Replace your epoch loop with this:

```python
best_score = -float("inf")
epochs_no_improve = 0

for epoch in range(1, max_epochs + 1):

    self.train_one_epoch(epoch)

    val_metrics = self.evaluate()
    current_score = compute_overall_score(val_metrics)

    print(f"Epoch {epoch} | Score: {current_score:.4f}")

    # --- Always run at least min_epochs ---
    if epoch <= min_epochs:
        if current_score > best_score:
            best_score = current_score
        continue

    # --- After min_epochs: apply early stopping logic ---
    if current_score > best_score:
        best_score = current_score
        epochs_no_improve = 0

        # Save best checkpoint
        self.save_checkpoint("best.pt")

    else:
        epochs_no_improve += 1

    # --- Stop condition ---
    if epochs_no_improve >= patience:
        print(f"Stopping early at epoch {epoch}")
        break
```

# 🚀 Optional — Weighted metric (better for your case)

Instead of simple average:

```python
def compute_overall_score(metrics):
    weights = {
        "emotion_score": 1.5,
        "ideology_score": 1.2,
        "narrative_score": 1.0,
        "bias_score": 1.0,
        "narrative_frame_score": 1.2,
        "propaganda_score": 0.8
    }
    return sum(metrics[k] * weights[k] for k in metrics) / sum(weights.values())
```
