# 🔧 ✅ FIX 1 — Drop Single-Class Emotion Columns

### Add this utility

### `src/utils/label_cleaning.py`

```python
import numpy as np

def remove_single_class_columns(labels, min_pos=1, min_neg=1):
    """
    labels: numpy array (N, C)
    returns: filtered_labels, kept_indices
    """
    labels = np.array(labels)

    pos = labels.sum(axis=0)
    neg = labels.shape[0] - pos

    valid_mask = (pos >= min_pos) & (neg >= min_neg)

    filtered = labels[:, valid_mask]

    return filtered, valid_mask
```
