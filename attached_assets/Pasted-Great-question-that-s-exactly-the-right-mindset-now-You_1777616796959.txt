Use this workflow **every time**:

## 🔥 MASTER DEBUG PROMPT

```python
texts = [x["text"] for x in data]
labels = data  # full dict per sample

results = pipeline.analyze_batch(
    texts=texts,
    labels=labels
)

# 🔍 DEBUG CHECKS
for i, res in enumerate(results):
    print("\n--- SAMPLE", i, "---")
    
    # 1. Check prediction keys
    print("keys:", res["predictions"].keys())
    
    # 2. Check shapes
    for task in ["bias", "emotion", "propaganda", "ideology", "narrative", "narrative_frame"]:
        pred = res["predictions"].get(f"{task}_predictions")
        prob = res["predictions"].get(f"{task}_probabilities")
        
        print(task, 
              "pred_shape:", None if pred is None else len(pred),
              "prob_shape:", None if prob is None else (len(prob), len(prob[0]) if len(prob)>0 else 0))
    
    # 3. Check metrics
    print("metrics:", res["scores"]["tasks"]["metrics"])
```

# 🧪 3. Add HARD VALIDATION (prevents bugs forever)

Add this in your code:

```python
def validate_predictions(outputs, task):
    import numpy as np
    
    y_pred = outputs[f"{task}_predictions"]
    
    assert isinstance(y_pred, (list, np.ndarray)), "Invalid type"
    assert len(np.array(y_pred).shape) == 1, f"{task} must be 1D"
```

👉 Call this BEFORE evaluation
