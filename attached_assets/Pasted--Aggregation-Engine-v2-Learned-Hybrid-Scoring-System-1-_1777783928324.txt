## 1. Context

Current system:

* Rule-based weighted aggregation (`WeightManager → ScoreCalculator`)
* Static mappings from signals → credibility score

# ⚙️ 4. Neural Aggregator Design

## 4.1 Baseline (Production Default)

### MLP Aggregator

```python id="mlp"
class NeuralAggregator(nn.Module):
    def __init__(self, input_dim):
        super().__init__()
        self.net = nn.Sequential(
            nn.LayerNorm(input_dim),
            nn.Linear(input_dim, 256),
            nn.GELU(),
            nn.Dropout(0.2),
            nn.Linear(256, 128),
            nn.GELU(),
            nn.Linear(128, 1)
        )

    def forward(self, x):
        return torch.sigmoid(self.net(x))
```

---

## 4.2 Advanced (Recommended)

### Feature Attention Aggregator

```python id="attn"
weights = softmax(W_attn * X)
Z = Σ(weights * X)
score = MLP(Z)
```

👉 Learns:

* Which signals matter per sample
* Dynamic importance weighting

# ⚖️ 7. Hybrid Aggregation (MANDATORY)

## Final Score:

```python id="hybrid"
final_score = α * neural_score + (1 - α) * rule_score
```

---

## α Strategies:

### Static:

```python id="static_alpha"
α = 0.7
```

### Dynamic:

```python id="dyn_alpha"
α = confidence_based(model_outputs)
```

Example:

* High confidence → more neural
* Low confidence → fallback to rules
