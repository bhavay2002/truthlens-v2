## 🔧 Prompt: Generate `features.md` for `src/features`

You are a senior ML systems engineer and feature engineering expert with strong experience in production pipelines.

Create a highly structured and technically precise document that enables any ML engineer or developer to:

* Fully understand how features are created and transformed
* Trace feature generation from input data to model-ready format
* Understand the reasoning behind each feature

* Safely extend or modify the feature pipeline

Analyze ALL files inside:
src/features/
