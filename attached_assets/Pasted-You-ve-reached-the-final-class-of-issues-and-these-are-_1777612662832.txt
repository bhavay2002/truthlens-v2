# 🚨 Problem 1 — `invalid value encountered in divide`

This is your **biggest real bug now**:

```text
invalid value encountered in divide
```

---

## 🧠 Root Cause

You are doing something like:

```python
scores = scores / scores.sum()
```

OR:

```python
scores = (scores - mean) / std
```

👉 But:

* `sum = 0` OR
* `std = 0`

So → division by zero → **NaN**

---

## ✅ Fix (MANDATORY)

### Safe normalization:

```python
eps = 1e-8

scores = (scores - scores.mean()) / (scores.std() + eps)
```

OR:

```python
scores = scores / (scores.sum() + eps)
```

---

## 🔥 Even better (production-safe)

```python
def safe_normalize(x):
    eps = 1e-8
    if x.std() < eps:
        return x  # skip normalization
    return (x - x.mean()) / (x.std() + eps)
```
