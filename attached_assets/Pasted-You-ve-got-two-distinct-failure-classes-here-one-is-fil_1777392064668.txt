# 🔴 2. `load_config()` Missing Argument

### Error

```
TypeError: load_config() missing 1 required positional argument: 'path'
```

### Root Cause

Your function signature is:

```python
def load_config(path):
```

But you are calling:

```python
config = load_config()
```

---

# 🔧 Fix (choose architecture)

## ✅ Option A — Pass Explicit Path (recommended)

In `main.py`:

```python
config = load_config("src/config/config.yaml")
```

OR better:

```python
from pathlib import Path

config_path = Path(__file__).resolve().parent / "src" / "config" / "config.yaml"
config = load_config(config_path)
```

---

## ✅ Option B — Add Default Path (cleaner API)

Modify function:

```python
def load_config(path=None):
    if path is None:
        from pathlib import Path
        path = Path(__file__).resolve().parent.parent / "config" / "config.yaml"

    with open(path, "r") as f:
        return yaml.safe_load(f)
```

Now your original call works:

```python
config = load_config()
```

## ✅ Option C — Config Injection (best for ML systems)

Pass config from CLI or environment:

```python
import os

config = load_config(os.getenv("CONFIG_PATH", "src/config/config.yaml"))
```
